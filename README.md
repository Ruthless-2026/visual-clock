# 🕐 Visual Clock - 可视化时钟

一个精美的模拟 + 数字双模式可视化时钟，纯 HTML/CSS/JS 实现。

## ✨ 特性

- **模拟表盘** — 精致的圆形表盘，带有时针、分针、秒针
- **数字时钟** — 玻璃拟态风格的数字时间显示
- **多层拖尾特效** — 秒针带有 6 层渐变彗星尾迹
- **渐变光环** — 表盘外围蓝紫渐变的旋转光环
- **小时刻度高亮** — 当前小时刻度自动金色发光
- **背景粒子** — 动态星空浮动粒子效果
- **毫秒级平滑** — 指针动画基于 `requestAnimationFrame`，包含毫秒偏移
- **中心脉冲动画** — 表盘中心点呼吸灯效果

## 🚀 使用方法

直接在浏览器中打开 `clock.html` 即可，无需任何依赖或构建步骤。

```bash
# 或使用任意静态服务器
npx serve .
```
